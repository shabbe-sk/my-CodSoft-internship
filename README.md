## CodSoft Java Programming Internship

This repository contains the Java projects I completed as part of my CodSoft Programming Internship. Each project focuses on core Java concepts such as OOP, control flow, input handling, and basic application logic.

Projects

1. ATM Interface

File: ATMinterface.java

A console-based ATM simulator that allows a user to:


Check account balance
Deposit money
Withdraw money (with balance validation)
Exit the menu


Concepts used: Object-Oriented Programming (separate BankAccount and ATM classes), input validation with Scanner, loops, and switch-case menu handling.

Sample flow:

Welcome to the ATM!
1. Check Balance
2. Deposit Money
3. Withdraw Money
4. Exit
Please select an option:


2. Number Guessing Game

File: numbergame.java

A number guessing game where the program randomly picks a number between 1 and 100, and the user has 7 attempts per round to guess it correctly. The game tracks the player's score across multiple rounds and lets them choose to keep playing.

Concepts used: Random class, loops, conditionals, score tracking, and replay logic.


3. Student Grade Calculator

File: studentgradecalculator.java

A simple program that takes marks for a given number of subjects, calculates the total and average percentage, and assigns a letter grade (A–F) based on the average.

Concepts used: Arrays, loops, conditional grading logic, and basic arithmetic operations.


Tech Stack

Language: Java
Tools: JDK, any Java-compatible IDE or terminal


Clone the repository:

git clone https://github.com/shabbe-sk/my-CodSoft-internship.git
